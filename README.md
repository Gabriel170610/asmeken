# Сайт для приёма заявок

Полнофункциональное веб-приложение для управления заказами с админ панелью.

## Установка и запуск

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn
- VS Code (опционально)

### Шаг 1: Клонирование проекта

```bash
git clone <repository-url>
cd project
```

### Шаг 2: Установка зависимостей

```bash
npm install
```

### Шаг 3: Переменные окружения

Проект уже имеет файл `.env` с необходимыми переменными для подключения к Supabase:

```
VITE_SUPABASE_URL=https://erhmujanmyknmzltmhlt.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

Если вам нужна собственная база данных Supabase, создайте её и обновите эти значения в файле `.env`.

### Шаг 4: Запуск сервера разработки

```bash
npm run dev
```

Сайт будет доступен по адресу: `http://localhost:5173`

## Использование

### Для клиентов

1. Откройте главную страницу
2. Выбирайте товары из меню и добавляйте в корзину
3. Меняйте количество товаров в корзине
4. Нажмите "Оформить заказ"
5. Заполните форму (имя, телефон, адрес)
6. Отправьте заказ

### Для администратора

1. Перейдите на `/admin`
2. Войдите с логином и паролем админа
3. На вкладке "Заказы" просматривайте поступающие заказы
4. На вкладке "Меню" управляйте товарами:
   - Добавляйте новые товары
   - Редактируйте цены и описания
   - Отключайте/включайте товары
   - Удаляйте товары

## Создание админ аккаунта

1. Откройте [Supabase Dashboard](https://app.supabase.com)
2. Перейдите в Authentication → Users
3. Создайте нового пользователя (email и пароль)
4. После создания, откройте пользователя и добавьте в поле `app_metadata`:

```json
{
  "role": "admin"
}
```

5. Сохраните изменения

Теперь вы можете войти на `/admin` с этими учётными данными.
